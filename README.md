# 🚀 AI Supply Chain Digital Twin & Control Tower

## 📌 Project Overview

This project is an **AI-powered Supply Chain Digital Twin System** that simulates real-world logistics operations using Artificial Intelligence, Machine Learning, Graph Algorithms, and Real-Time Event Streaming.

It acts as a **real-time control tower** for monitoring shipments, predicting delays, optimizing suppliers, and detecting global disruptions.

---

## 🧠 Key Features

### 📦 1. Digital Twin Simulation
- Real-time shipment tracking
- Multi-city logistics network
- Live movement visualization using PyDeck

---

### ⚠️ 2. Global Risk Intelligence
- News-based disruption detection (war, strike, flood, oil crisis)
- Kafka-style event stream simulation
- Dynamic risk scoring system

---

### 🤖 3. AI Decision Engine
- Google Gemini AI integration
- Risk analysis and recommendations
- Natural language supply chain assistant

---

### 📊 4. Machine Learning Prediction
- Linear Regression model
- Predicts shipment delay based on:
  - Volume
  - Global risk score
  - Random real-world noise simulation

---

### 🏭 5. Supplier Optimization Engine
- Weighted scoring system based on:
  - Cost
  - Reliability
  - Risk
  - Delay
- Automatically selects best supplier

---

### 🛣 6. Route Optimization System
- Graph-based shortest path algorithm (NetworkX)
- Simulates real-world logistics routing

---

### 🌍 7. Live Map Visualization
- PyDeck-powered interactive map
- Shipment movement animation
- Disruption zone overlay

---

## 🏗 System Architecture
User Input → Streamlit UI
↓
Data Layer (Pandas)
↓
Risk Engine + Kafka Simulation
↓
ML Model (Delay Prediction)
↓
Graph Optimization (NetworkX)
↓
AI Layer (Gemini API)
↓
Visualization (Plotly + PyDeck)
